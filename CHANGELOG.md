# Changelog

All notable changes to DiTiS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).
Version format: YYYYMMDD (date-based versioning).

## [20260222] - 2026-02-22

### Added
- 🌐 **紹介ウェブサイト**: ランディングページを作成
- ⌨️ **ヘルプ強化**: Space, F2, ドット, W/A/S/D, Ctrl+Shift+E, Ctrl+I のショートカットをヘルプに追加
- 📂 **最近使用したファイルを開く**: ファイルメニューに最近使用したファイルのサブメニューを追加（最大10件、LocalStorage保存）
- 🔄 **起動時にシート状態を復元**: 編集メニューに「起動時に前回のシート状態を復元する」オプションを追加
- ✖️ **タブに×ボタン追加**: 各タブにクリックで閉じられる×ボタンを追加（確認ダイアログあり）
- 🎬 **AEキーフレームデータコピー**: 列ヘッダーを右ダブルクリックでその列のAE形式キーフレームデータをクリップボードにコピー
- ⚙️ **キーフレームデータバージョン変更**: 編集メニューからコピーするキーフレームデータのAEバージョンを変更可能（デフォルト: 9.0）

### Changed
- 🎨 **デフォルトテーマ変更**: 初期テーマをライトテーマからグリーンテーマに変更
- 🔤 **フォントサイズ簡素化**: 計算式から5段階×5種類の固定値テーブルに変更
- 📐 **CSSデフォルト値統一**: CSSのフォントサイズデフォルトをchangeFontSize(12)の出力値と一致するよう修正
- 🏷️ **バージョン表記**: v2026.02.22（betaを削除）
- 📝 **ウィンドウタイトル**: 「DiTiS - v2026.02.22」に変更
- 💾 **保存時シート名更新**: ファイル保存時にファイル名（拡張子なし）をシート名・タブに反映
- 🗂️ **選択状態の保存・復元**: シート切り替え時に各シートの選択状態を保存・復元、新規作成時はA1を選択

### Fixed
- 🐛 **初期選択修正**: 起動時のA1セル選択が確実に動作するよう修正
- 🐛 **選択維持修正**: 再レンダリング後にセル選択が消える問題を修正
- 🐛 **直線区間編集修正**: 縦線区間内でのセル編集時に正しくハイフン表示に切り替わるよう修正
- 🐛 **+/-キー表示修正**: +/-キー入力後に即座に表示が更新されるよう修正（キャッシュ再計算追加）
- 🐛 **列間ヘッダー背景色修正**: 挿入行の列間フレームヘッダー背景色が正しく反映されるよう修正
- 🐛 **ヘルプ中のキー入力ブロック**: ヘルプ表示中にF1/Esc以外のキー入力を無効化
- 🐛 **ヘルプ×項目削除**: 未実装だった「X: ×を入力」をヘルプから削除
- 🐛 **PageUp/PageDown削除**: 未実装だったPageUp/PageDownをヘルプから削除

---

## [20260211] - 2026-02-11

### Added
- 📐 **極小表示サイズ**: 5段階目の表示サイズ「極小」を追加（セル幅30px × 高さ8px）
- 📊 **ダイアログ比較表示**: 尺変更・列数変更・フレーム挿入ダイアログに「現在 → 変更後」のリアルタイム比較を表示
- 📋 **列間フレームヘッダー**: 列間にフレームヘッダーを表示するトグル機能
- 🆕 **新規シートダイアログオプション**: 新規シート作成時にダイアログを表示するオプション
- 🔄 **自動アップデート機能**: GitHub Releasesを使った自動アップデート
  - 起動時に自動チェック（1日1回まで、オン/オフ切替可能）
  - ヘルプメニューから手動で更新確認可能
  - 更新ダイアログで「今すぐダウンロード」「後で通知」「この更新を無視」を選択可能
  - 署名検証によるセキュアな更新
- ✅ **テストスイート拡充**: 315テスト（11ファイル）
  - Undo/Redo 履歴管理テスト
  - フレーム操作・クリップボード・状態管理テスト
  - キーボードショートカットテスト
  - メニューイベントテスト
  - 自動アップデートテスト
  - 多言語対応テスト

### Changed
- 🗂️ **表示メニュー再整理**: 論理的なグループ分けに再構成
  - シート表示設定（コマ表示・ヘッダー表記・表示サイズ・列間ヘッダー）
  - 表示設定をリセット
  - 外観（テーマ・言語）
  - オプション（常に前面に表示・自動スクロール）
  - ページを再読み込み
- 🗂️ **シートメニュー再整理**: 新規シートダイアログオプションを表示メニューからシートメニューの「シート設定」直下に移動
- 🔤 **メニュー名称変更**: 「表記モード」→「ヘッダー表記」、「中間フレーム番号」→「列間にフレームヘッダーを表示」
- 🏗️ **HTML メニュー削除**: Tauriネイティブメニューに完全移行
  - HTMLツールバー（~114行）、メニューハンドラ（~700行削減）、CSS（~140行）を削除
  - handleMenuEvent を直接関数呼び出しに変更
- 📌 **バージョン管理方式変更**: セマンティックバージョニングから日付ベース（YYYYMMDD）に変更
- 🎨 **列間ヘッダーの配色**: 左ヘッダーと同じ背景色に統一
- 📝 **セルフォントサイズ統一**: 標準以下の表示サイズでセル内フォントサイズを15pxに統一

### Fixed
- 🐛 **Enterキー選択表示**: 単一セル選択時のEnterキーで選択が表示されない問題を修正
- 🐛 **Tauri IPCパラメータ**: camelCaseパラメータ送信に修正
- 🐛 **伸ばし棒マーカー**: AE送信時の伸ばし棒のマーカー尺を修正
- 🐛 **表示サイズ変更エラー修正**: `cellWidth is not defined` エラーを修正
- 🐛 **メニュー再構築エラー修正**: 言語切替時の `rebuildMenu` パラメータ不足エラーを修正
- 🐛 **Tauri IPCパラメータ修正**: `rebuildMenu` のパラメータをcamelCaseに統一（Tauri 2.x準拠）
- 🐛 **i18nキー追加**: `menu.file.saveAs` の翻訳キーを追加
- 🐛 **入力フォーム自動補完無効化**: ダイアログの入力欄でブラウザの自動補完ポップアップを非表示に
- 🐛 **単一セル選択時のEnterキー選択表示**: 単一セル選択でEnterキー押下後に選択範囲が表示されない問題を修正

## [0.0.3] - 2026-02-07

### Added
- 🎯 **自動プリコンポーズ**: AE送信時にコンポジションでないレイヤーを自動でプリコンポーズするオプション
- 🔤 **全角アルファベット対応**: AEレイヤー名マッチング時に全角アルファベットを半角に自動変換
- 🔍 **先頭アルファベットマッチング**: レイヤー名の先頭アルファベットでシート列名を自動マッチング（例: "d 2" → "D"）

### Fixed
- 🐛 **マーカー尺修正**: 最後のマーカー（伸ばし棒が最後まで続く場合）の尺が1フレーム分しかなかった問題を修正

### Changed
- 🎨 **マーカーオプションのパネル化**: マーカー追加チェックボックスをパネルでグループ化
- 🔄 **レイヤー選択状態の復元**: AEスクリプト実行後にレイヤーの選択状態を実行前に復元

---

## [0.0.2] - 2026-02-03

### Added
- 📂 **XDTS形式対応**: XDTS（XML Digital TimeSheet）形式の読み込み・保存に対応
- 💾 **TDTS形式保存**: TDTS（東映デジタルタイムシート）形式での保存に対応
- ➖ **ホールド自動挿入**: ファイル読み込み時にホールド（"-"）を自動挿入
- 📝 **名前を付けて保存**: Ctrl+Shift+S で常に保存ダイアログを表示
- 🎯 **マーカー追加オプション**: AE送信時にタイムリマップ値をレイヤーマーカーとして追加可能
- ⚙️ **AE設定保存**: 空セル処理方法とマーカー追加オプションをAE環境設定に保存

### Changed
- 🗂️ **メニュー整理**: ファイルメニューから不要な個別読み込み/保存メニューを削除し、シンプルに統一

---

## [0.0.1] - 2026-01-22

### Added
- 🎨 **多言語対応**: 日本語・英語の完全対応（UI、メニュー、ステータスバー、ExtendScript出力）
- 📋 **メニュー状態の永続化**: アプリ再起動時に前回のテーマ、フレームフィルター、ヘッダーモード、フォントサイズ、言語設定を自動復元
- 🔄 **言語切り替え機能**: メニューから即座に言語を切り替え、すべてのUI要素が更新
- 🎯 **「表示設定をリセット」メニュー**: ビューメニューから表示設定を初期状態に戻す機能
- 🌈 **3つのテーマ**: ライト、ダーク、グリーンテーマを選択可能
- 📊 **フレームフィルター**: 全フレーム、奇数フレーム、偶数フレームの表示切り替え
- 📑 **ヘッダー表示モード**: 詳細モード（A, B, C...）とシンプルモード（1, 2, 3...）
- 🔤 **フォントサイズ調整**: 小、標準、大、特大の4段階
- 🐛 **デバッグモード**: 開発者向けの詳細ログ表示
- 📌 **常に最前面表示**: ウィンドウを常に最前面に固定
- 💾 **自動保存**: LocalStorageを使用した設定の自動保存
- 🎬 **After Effects連携**: ExtendScript(.jsx)形式で出力、または直接送信
- 🔍 **スマートなレイヤー検出**: 未選択時は全AVLayerを対象に自動適用
- ⚡ **複数AE対応**: 複数のAfter Effectsが起動中の場合、最前面のインスタンスに送信
- ⚠️ **FPS警告**: コンポジションとシートのフレームレートが一致しない場合に警告
- 📁 **柔軟なファイル保存**: JSON/STS形式を選択可能、上書き保存対応
- ⏮️ **Undo/Redo**: 50段階の操作履歴
- 🔁 **ループ選択機能**: 右クリックメニューから選択範囲を自動で繰り返し（アニメーションパターンに便利）
- 📊 **複数シート対応**: タブで複数のタイムシートを管理（ドラッグ&ドロップで並び替え可能）
- ⌨️ **最適化されたキーボードショートカット**: 高速入力をサポート
- 🎯 **直感的なExcelライクUI**: スプレッドシートインターフェース

### Fixed
- ✅ 言語切り替え時にTauriネイティブメニューが更新されない問題を修正
- ✅ 言語切り替え時にステータスバー（Ready、Time、セル情報）が翻訳されない問題を修正
- ✅ アプリ起動時にメニューのチェックマークが現在の設定と一致しない問題を修正
- ✅ Tauri IPCパラメータ命名の問題（camelCase vs snake_case）を修正
- ✅ メニューイベント処理での重複したチェックマーク更新を削除
- ✅ 排他的チェックボックス処理の安定性を向上
- ✅ 列削除時に確認ダイアログが表示される前に削除される問題を修正（Tauri非同期ダイアログを使用）
- ✅ インストール後初回起動時にA1～B26が誤選択される問題を修正（layer IDの型不一致を解消）

### Technical
- 🔧 **Tauri 2.9.5**: 最新のTauriフレームワークを使用
- 🦀 **Rust backend**: メニュー管理とシステム統合
- 📦 **NSIS installer**: Windowsインストーラー対応
- 🎨 **CSS Variables**: テーマシステムの実装
- 🌐 **i18n system**: 翻訳管理システム
- 💾 **LocalStorage**: 設定の永続化
- 🔄 **State management**: アプリケーション状態管理
